🔗 Knot SDK Test Page

Use this page to test the Knot Web SDK with sessions created via the terminal scripts.

📋 Instructions:

  1. Run ./create-session.sh USER_123 production in terminal
  2. Copy the session ID and paste it above, OR click "Load Last Session"
  3. Select the correct environment
  4. Click "Initialize Knot SDK"
  5. Complete the account linking process
  6. Check your webhook server for the NEW_TRANSACTIONS_AVAILABLE event
  7. Run ./sync-transactions.sh [merchant_id] USER_123 production
